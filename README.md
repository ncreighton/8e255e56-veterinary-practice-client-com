== Description ==

A WordPress plugin for veterinary clinics and animal hospitals managing client communications and appointment workflows. Features automated vaccination expiration alerts, appointment reminder SMS and email sequences, online booking with species-specific triage questions, medical record access portals for clients, and post-visit care instruction delivery. Reduces no-shows by 40 percent through multi-touch reminder sequences and two-way text confirmation. Integrates with Stripe for payment deposits and AVImark for practice management.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Veterinary Practice Client Communications and Appointment Reminder WordPress Plugin
